New Minecraft Plugin

About a month ago now I started developing another Minecraft plugin. This one is, at least tentatively, called "Challenge Mode", and is meant to manage different 'survival with a twist' game modes. You can download the plugin from GitHub Right now, the plugin contains two 'modes': SupplyDrop and BorderShrink. BorderShrink just shrinks the world border for now, and is meant as something other modes can build on. SupplyDrop works off of BorderShrink, adding randomly generated chests containing loot. You can configure the timings of the chests appearing and the contents of the chest through the config.yml. I've also incorporated a scoring system. Every chest contains a Gold Nugget called Tokens with special metadata so that it cannot be replicated by a player. If you use this plugin for an event, you can use these items to score the players. On my server, I will be asking players to screenshot their inventories at the end of the game to show how many of these Tokens they have collected.
